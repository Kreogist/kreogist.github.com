Download Resources

Here are some resources for you to customize your μ.

Images

  • Genre Images (37.6MB)

    • Download links: SourceForge, Baidu

    You can always add, remove or modify any genre image in your music library folder. To find your genre image folder, you can find it at General tab in Preference.
    For default case, you should extract these images to:

    • Windows: Your Documents/Kreogist/Mu/Library/Music/Genres
    • Mac OS X: /home/Documents/Mu/Library/Music/Genres
    • Linux: ~/.kreogist/mu/Library/Music/Genres

    If you have moved the library folder, put it in the /Library/Mu/Genres folder in you library.
    Restart μ, and μ will load all the genre images.

Fonts

    You can always add, remove or modify any customize fonts in your library folder. μ will automatically load these fonts every times initiallize.
    For default case, you should put these fonts to:

    • Windows: Your Documents/Kreogist/Mu/Fonts
    • Mac OS X: /home/Documents/Mu/Fonts
    • Linux: ~/.kreogist/mu/Fonts

    If you have moved the library folder, put it in the /Library/Mu/Fonts folder in you library.
    Restart μ, and μ will load all the fonts.
