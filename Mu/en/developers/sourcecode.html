Open Source

We built μ based on many projects created by the open source community, and we learnt a lot from these open source projects. We hope to learnt more and helped others to avoid some tricky problems we faced, and we want to join the open source community to be a member of them. We believes that using open source methodology could make our project much more steady. So, we released μ's source code and all the documentation.

μ, μ's source code and μ's SDK are available under The GNU General Public License 2.0 and DO WHAT THE FUCK YOU WANT TO PUBLIC LICENSE 2.0. To learn more about license using in μ, please visit the license section.

Get the Sources

You can always download the latest code from our homepage of the project hosted on Github.

You can also get source code snapshot packages from below. The source code is also available as a single tar.gz file for Mac OS X/Linux users or zip file for Windows users.
